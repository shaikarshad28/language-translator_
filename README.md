 Indian Language Translation Tool

A simple and user-friendly web application that translates text between Indian languages using a Translation API.


Features

- Translate text instantly
- Supports Indian languages
- Copy translated text
- Text-to-Speech support
- Simple and responsive UI



Supported Languages

- English
- Hindi
- Kannada
- Malayalam
- Bengali
- Gujarati
- Punjabi



Technologies Used

- HTML
- CSS
- JavaScript
- MyMemory Translation API
